# Dealers

<SplitColumn>
  <div>
    This object represents a dealer profile created by a user. Includes business
    information and submitted applications.
  </div>
  <RestList>
    <RestElement
      method="POST"
      path="/api/v0/dealers/search"
      anchor="search-dealers"
    />
  </RestList>
</SplitColumn>

### Search dealers

<SplitColumn>
  <div>
    <p>
      Search for dealers using a variety of filters. A maximum of 100 results
      will be returned per request. If additional results are available, they
      can be retrieved using the <code>nextPage</code> parameter from the
      previous response.
    </p>
    <h4>Authentication</h4>
    <div>
      <Link to="/docs/api/gettingstarted/oauth">Service Integration</Link> with{" "}
      <code>vendor:read</code> scope.
    </div>
    <div style={{ marginBottom: "1.5rem" }} />
    <h4>Request</h4>
    <Attribute id="su-nextPage" name="nextPage" type="string" optional>
      A cursor for pagination across multiple pages of results. Don't include
      this attribute on the first call. Use the <code>nextPage</code> parameter
      from the previous response for each subsequent request.
    </Attribute>
    <Attribute id="su-limit" name="limit" type="number" optional>
      The maximum number of results to return, between 1 and 100. Defaults to
      100.
    </Attribute>
    <ParentAttribute id="su-filter" name="filter" type="object">
      <span>Filters to apply to the search.</span>
      <Attribute id="su-flr-userIds" name="userIds" type="Array of string">
        The IDs of the users to filter by.
      </Attribute>
    </ParentAttribute>
